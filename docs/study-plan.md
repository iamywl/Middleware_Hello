## 학습 팁

1. **실습 우선**: 문서를 읽기만 하지 말고, 반드시 `docker-compose up`으로 환경을 띄우고 직접 확인해야 한다. 눈으로 읽는 것과 손으로 해보는 것은 완전히 다르다.
2. **노트 필수**: 각 주차 마지막 날에 핵심 개념을 자기 말로 정리해야 한다. 남에게 설명할 수 있어야 진짜 이해한 것이다.
3. **연결 사고**: "이 설정이 바뀌면 모니터링에 어떤 영향이 있을까?" 같은 방식으로 컴포넌트 간 연결을 항상 의식하는 것이 중요하다.
